CT Scores (AutoPR testing)

Features:

INSTAGRAM/FACEBOOK API CALLS

- Used to access Instagram and Facebook pages. Necessary for automated posting.
- Handles captions, tagging, sharing.
- Posts carousels with finished posts.

CLOUDFLARE API CALLS

- Used for temporary image hosting. Instagram can only post photos that are publicly accessible.
- Create a bucket where images ready to be posted can be made publicly accessible, allowing Instagram to find photo/s.
- Delete photos from bucket once Instagram has finished uploading the post.

PILLOW PYTHON LIBRARY

- Edit an existing score template as a PSD per user input. Exports a jpeg for posting.
- Manipulate text layers (identified by name) for scores, records, rankings, etc.
- Repeat process for a number of images, then send images to Cloudflare R2 to be accessed by Instagram.

GOOGLE SHEETS API

- Use a Google Form to get user input. Link the Google Form to a Google Sheet.
- API allows code to access the Google Sheet so that its contents can be stored in GitHub repo.
- Provides data for Instagram posts and website.

WEBSITE (GITHUB PAGES)

- Web page serves as another place for people to get score information.
- Uses the same CSV file as the python code to get info.
- GitHub actions and workflows update scores every 20 mins from 8 AM to 11 PM EST.
